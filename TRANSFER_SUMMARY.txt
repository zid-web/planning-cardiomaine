💡 TIPS IMPORTANTS
================================================================================

1. Tous les imports doivent utiliser le préfixe @/
   ✅ import { Button } from "@/components/ui/button"
   ❌ import { Button } from "../components/ui/button"

2. Server Actions pour la BD (côté serveur sécurisé)
   → Fichier: /app/actions/schedule-actions.ts

3. Real-time Supabase configuré automatiquement
   → Le planning se met à jour en live

4. PWA déjà configurée
   → Manifest.json dans /public

5. Tailwind v4 avec @theme dans globals.css
   → Pas besoin de tailwind.config.js
