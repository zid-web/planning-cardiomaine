🔧 INSTALLATION RAPIDE
══════════════════════════════════════════════════════════════════════════════

1. Créer la table dans Supabase:
   → Fichier: vacations.sql
   → SQL Editor → New Query → Copier/Coller → Run

2. Lancer l'application:
   → npm run dev

3. Accéder aux vacations:
   → Cliquer sur "✈️ Vacances" dans la barre admin

4. Générer les gardes:
   → Ajouter une vacation
   → Cliquer "Générer Gardes Nuit"
   → Les médecins en vacations seront exclus

🗄️ ARCHITECTURE BASE DE DONNÉES
══════════════════════════════════════════════════════════════════════════════

Table: doctor_vacations

Colonnes:
  - id: UUID (PRIMARY KEY)
  - doctor_id: TEXT (médecin)
  - start_date: DATE (début)
  - end_date: DATE (fin)
  - reason: TEXT (raison)
  - created_at: TIMESTAMP
  - updated_at: TIMESTAMP

Contraintes:
  - UNIQUE(doctor_id, start_date, end_date)

Index:
  - idx_doctor_vacations_doctor_id
  - idx_doctor_vacations_dates

RLS Policies:
  - SELECT: Utilisateurs authentifiés
  - INSERT: Utilisateurs authentifiés
  - UPDATE: Utilisateurs authentifiés
  - DELETE: Utilisateurs authentifiés
